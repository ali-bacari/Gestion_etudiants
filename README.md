# Application de bonjour avec nodejs et express
## Application de bonjour développée avec nodejs et express. 

## technologies

- nodejs
- express

## Structure du projet

 Gestion_etudiants/
  server.js
  package.json
  package-lock.json

## Prerequis

- nodejs version 18+

## Verification

- node -v
- npm -v

## Installation

### 1. cloner le depot
git clone https://github.com/ali-bacari/Gestion_etudiants.git

### 2. se deplaser dans le dossier
cd Gestion_etudiants.git

### 3. instalation des dependances

npm install

### 4. lancer le projet
node server.js
